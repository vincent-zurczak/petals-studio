Read Me
=======

This project was created by Petals Studio.
* Make sure you have installed Maven.
* Open a command line interface in the project directory.
* Type in "mvn eclipse:eclipse".
* Make sure you have defined the M2_REPO variable in the studio 
  (Window > Preferences > Java > Classpath Variables and make it point to your local Maven repository).
* Refresh the project in the studio.

Alternatively, you can install m2eclipse to automatically resolve Maven dependencies in your Eclipse.
Complete the code skeleton and keep on using Maven to compile your component.

A developer guide for Petals components is available at 
https://doc.petalslink.com/display/petalscomponents/Components+Developer%27s+Guide
